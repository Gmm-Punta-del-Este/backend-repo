Apartamento
TYPE CONTROLLER MAPPER ENDPOINT VARIABLES Descripcion EJEMPLO
GET /apto /listar / NONE Retorna lista de los apartamentos del sistema ip:port/apto/listar
GET /apto /listar /libres NONE Retorna lista de los apartamentos libres del sistema ip:port/apto/listar/libres
GET /apto / {nroapto} Retorna la informacion del apartamento con {nroapto} del sistema ip:port/apto/{nroapto}
GET /apto /listar /propietario /{documento} Retorna lista de los apartamentos que pertenecen al propietario con {documento} del sistema ip:port/apto/listar/propietario/{documento}
POST /apto / NONE NONE Ingresa un nuevo apartamento al sistema, chequeaando el mismo esta activo (consultar esto)

Ejemplo:

{ "propietario":"1234", "apto": { "nroApto": 102, "notas": "hola", "venta": false, "alquilable":false }, "user":{ "login":"jch321", "password":"1234" } } 		ip:port/nota/
DELETE /apto / NONE {id} Borra el apartamento con {ID} del sistema ip:port/apto/{id}
