Usuario
TYPE CONTROLLER MAPPER ENDPOINT VARIABLES Descripcion EJEMPLO
GET /persona /usuario /listar NONE Nos trae una lista de todos los usarios en el sistema ip:port/persona/usuario/listar
GET /persona /usuario /activos NONE Nos trae una lista de todos los usarios en el sistema ACTIVOS ip:port/persona/usuario/inactivos
GET /persona /usuario /inactivos NONE Nos trae una lista de todos los usarios en el sistema INACTIVOS ip:port/persona/usuario/activos
GET /persona /usuario / {username} Retorna el usuario por el {username} ingresado ip:port/persona/usuario/tuperrita
GET /persona /usuario / {documento} Retorna el usuario por el {documento} ingresado ip:port/persona/usuario/123123
GET /persona /usuario / {username}/{password} Retorna el usuario con el {username} y {password} ingresado ip:port/persona/usuario/tuperrita/panqueque
POST /persona /usuario / NONE Ingresa un usuario desde el JSON Enviado.
Ejemplo: { "documento":"123123", "nombre":"Cooso", "apellido1":"Ba", "apellido2":"Cro", "login":"jch321", "tipoUsuario":"User", "password":"1234", "direccion":"Tiro tiro punialada22" } 		ip:port/persona/usuario/
PUT /persona /usuario /estado NONE Modifica el estado de un usuario a traves de un Json. ( Nota 1 )
Ejemplo: { "documento_usuario":"123123", "flag":"desactivar" } 		ip:port/persona/usuario/estado
